Open Sourcery

All of our GitHub repositories are public at DoSomething.org. Below is a showcase of a select few, all built on top of open source software.

  • dosomething

    A Drupal 7 distribution that powers the DoSomething.org web platform.

  • neue

    Pattern library and base styles for DoSomething.org's web interfaces.

  • ds-homestead

    Laravel Homestead local development environment tweaked for DoSomething developers.

  • voting-app

    A voting application written in Laravel that currently runs Celebs Gone Good.
