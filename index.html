Open Source

All of our work is built on open-source software. We stand on the shoulders of giants (Linux, PHP, Drupal, Laravel, Node, MySQL, MongoDB, jQuery...) and consider it our responsibility to give back.

Here's a selection of open-source packages we've released:

  • vision See where your code has been deployed and whats been tested at a glance.
  • npm-notifier Send notifications to Slack when a package is updated.
  • count-von-count Parse written numbers into digits.
  • drupal-external-auth Base module for using external APIs for Drupal authentication.
  • hubot-shakeshack A Hubot script that posts a photo of the Shake Shack cam.
  • stathat-php Simple, modern PHP API wrapper for StatHat.

We also write all of our code in the open. Our production code base & internal tools are open-sourced for inspection, improvement, and forking. Feel free to take a look at any of our projects (and if you like what you see, consider joining the team!)

  • Phoenix Our production website, powered by Drupal.
  • Forge Interface framework and pattern library.
  • Northstar User & identity service, built with Laravel.
  • Aurora User admin tool. This is the human-facing frontend to Northstar.
  • Quicksilver Message broker, built with RabbitMQ and Sails.
  • Rogue Campaign activity service, built with Laravel.
  • Gateway An internal API client with simple authentication built-in.

(And yes, they're all named after superheroes.)
